Web Programming Coursework

Here is a list of some of the projects and mini-projects I completed for my Web Programming course (taught by Ming Chow):
  • Security Assessment

    Write-up describing results of pen testing a peer's web application.

  • Landmarks

    Wrote client interface and web server for a web application that uses Google Maps Api to plot the client's location, the locations of historic landmarks within 1 mile of the client, and the locations of other users who have sent their locations to the server.

  • Fresh Fridge App

    Group project. Keeps track of expiration dates of client's perishable goods, sends email notifications when foods are nearing expiry, makes Api calls for recipe recommendations, uses data visualizations to show user trends of food waste and consumption.
